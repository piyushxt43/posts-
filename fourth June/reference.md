# Carousel Style Reference (Terracotta Plugin Deck)

Public style notes for the fourth June batch. No local asset paths.

## Format

- 4:5 or 3:4 vertical
- 7-8 slides per carousel
- Comment keyword → PDF lead magnet

## Visual DNA

| Element | Spec |
|---------|------|
| Background | Terracotta `#C45E3B`, paper grain |
| Header | `JUN @2026  PIYUSH.GLITCH  [slide N / 08]` |
| Tags | Black rounded pill, white text |
| Cards | White GitHub-style repo cards, tilted, tape corners |
| Headlines | Huge bold lowercase sans, gradient emphasis words |
| Footer | Progress dots, `@piyush.glitch` |

## Slide roles

1. **Cover** — hero hook, gradient title, statue + floating app icons
2. **Plugin cards** — black pill tag + repo card with tape
3. **Bonus slides** — checklist or workflow tiles
4. **CTA** — comment keyword for PDF
