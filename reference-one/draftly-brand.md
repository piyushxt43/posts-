## Two paths

1. **3D Builder** (`draftly.space/3d-builder`) — blank canvas, full prompt pipeline
2. **Presets** (`draftly.space/presets`) — 47 production-ready sites, click-to-edit, chat customize

---

## 5-step pipeline

1. **Prompt Input** — describe the website
2. **Visual Draft** — AI still image
3. **Motion Pass** — ~8s video
4. **Website Build** — frame extraction (36–2000 frames)
5. **Website Ready** — scroll-synced HTML

Preset path adds **Pick a preset** before Describe.

---

## Four prompt fields (builder)

| Field | Purpose |
|-------|---------|
| Website prompt | Full site brief: nav, sections, copy, scroll behavior |
| Image prompt | Hero still, 16:9, photoreal art direction |
| First + last frame | Animation bookends for continuity |
| Video prompt | 8s motion loop for frame extraction |

After ship: **chat** to change layout, copy, colors, media.
